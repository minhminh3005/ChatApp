hatApp

A real-time chat application built with React, TypeScript, and Node.js. This project features instant messaging and is designed for future AI integration.

## Features

- Real-time chat between users
- Modern React frontend
- Node.js/Express backend
- TypeScript for type safety
- Upcoming: AI-powered chat features

## Getting Started

### Prerequisites

- Node.js (v16+ recommended)
- npm

### Setup

#### 1. Clone the repository

```bash
git clone https://github.com/minhminh3005/ChatApp.git
cd chatapp
```

#### 2. Install dependencies

##### Frontend

```bash
cd frontend
npm install
```

##### Backend

```bash
cd ../backend
npm install
```

#### 3. Start the application

##### Backend

```bash
npm run start
```

##### Frontend

Open a new terminal:

```bash
cd frontend
npm start
```

The frontend will run on `http://localhost:5173` and the backend on `http://localhost:5001` (default).
